# Login Page UI

This project is a simple and clean **Login Page User Interface** built using HTML and CSS. It includes email and password input fields, a terms and conditions checkbox, and social media login icons.

## Project Purpose
- Practice building form-based UI components
- Create a modern and minimal login card design
- Improve HTML and CSS layout skills

## Features
- Centered login card layout
- Email and password input fields
- Terms of Service and Privacy Policy agreement section
- Login button
- Social media login icons (Google, Facebook, Twitter, Apple)
- Responsive and clean design

## Technologies Used
- HTML5
- CSS3
- Font Awesome (for social media icons)

## Project Structure
- **index.html** — Main HTML file for the login page
- **CSS/main.css** — Stylesheet for layout and design
- **README.md** — Project documentation

## How It Works
The login page is wrapped inside a centered container using Flexbox.  
The card component contains form fields, a checkbox for terms agreement, a login button, and social media icons styled with Font Awesome.

## Advantages
- Simple and easy-to-understand structure
- No JavaScript required
- Easy to customize colors, fonts, and layout
- Suitable for beginners practicing UI design
